UNIX 기초 명령어
===


* 디렉토리 내용의 목록(list)을 보는 명령어 : __ls__
> ```
> ls
> 
> ```

* 디렉토리를 바꿈(change directory) : __cd__
> ```
> cd               -> 홈 디렉토리로 이동
> cd [목적 디렉토리]   -> [목적 디렉토리]로 이동
> 
> ```

* 디렉토리 만들기 : __mkdir__
> ```
> mkdir [디렉토리 이름]
> 
> ```

* 파일 지우기 : __rm__
> ```
> rm [파일 이름]
> 
> ```

* 파일 복사하기 : __cp__
> ```
> cp [파일명][목적지]
> 
> ```

* 파일 옮기기: __mv__
> ```
> mv [파일명][목적지]
> 
> ```
