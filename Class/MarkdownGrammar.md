Markdown
===

---
* 마크다운(markdown)은 일반 텍스트 문서의 양식을 편집하는 문법
* README 파일이나 온라인 문서, 또는 일반 텍스트 편집기로 문서 양식을 편집할 때 쓰임.
* 마크다운을 이용해 작성된 문서는 쉽게 HTML등 다른 문서형태로 변환이 가능

* 인용문

	```{.python}
	> 인용문
	> > 연속 인용문
	```

* __순서가 있는 리스트__

	```{.python}
	1. 첫번째
	2. 두번째
	3. 세번째
	```
	1. 첫번째
	2. 두번째
	3. 세번째

* __순서가 없는 리스트__

	```{.python}
	* 첫번째
	+ 두번째
	- 세번째
	```
	* 첫번째
	+ 두번째
	- 세번째

* __혼합 리스트__

	```{.python}
	1. 첫번째
	   +  두번째
	2. 세번째
	```
	
	1. 첫번째
   		+  두번째
